> Useful oneliners and shorthand functions



## Usage

### Install

```
$ npm install --save 1-liners
```

### Usage in es5

```js
var map = require('1-liners').map;
var map = require('1-liners/map');
```

### Using in es6

```js
import map from '1-liners/map';
import map from '1-liners/module/map'; // uncompiled
import { map, filter } from '1-liners';
```
